# CS Construct Mode

CS:GO-стиль геймод для Garry's Mod — командный режим T/CT с экономикой, закупкой оружия, ботами и системой раундов.

---

## Зависимости (обязательно установить через Workshop)

Подпишись на все аддоны из коллекции:
**https://steamcommunity.com/sharedfiles/filedetails/?id=3691972773**

---

## Установка

### 1. Установить зависимости
Подпишись на все аддоны из коллекции выше через Steam Workshop.

### 2. Скачать геймод
Нажми **Code → Download ZIP** на этой странице и распакуй папку `cs_construct_mode` в:
```
GarrysMod/garrysmod/addons/
```

### 3. Запустить
- Открой Garry's Mod
- Создай новую игру
- Выбери геймод **CS Construct Mode**
- Выбери любую карту (рекомендуется `de_dust2` или `cs_office`)

---

## Управление

| Клавиша | Действие |
|---------|----------|
| `F4` | Меню настроек |
| `B` | Меню закупки |
| `1–5` | Слоты оружия |
| `Shift` | Тихий шаг |
