# CS Construct Mode

CS2-стайл геймод для Garry's Mod на основе SWCS-оружий.

**Особенности:**
- Раунды T vs CT в стиле CS2
- Закупка оружия (F4-меню / B)
- Система бомбы: случайному игроку T даётся C4, победа по взрыву или разминированию
- Таймер переключается на обратный отсчёт бомбы после закладки
- Killfeed с иконками оружий и маркером хедшота (☠)
- Движение в стиле CS:GO (скорость зависит от оружия, антибхоп, тихий шаг Shift)
- Броня и набор сапёра для CT
- Выбор модели игрока и перчаток через F4

## Установка (пошагово)

### Шаг 1 — Подпишись на Workshop-аддоны
Зайди по каждой ссылке из таблицы выше и нажми **Subscribe**. Запусти GMod хотя бы раз, чтобы они скачались.

### Шаг 2 — Смонтируй CS:GO в Garry's Mod
1. Запусти Garry's Mod
2. Главное меню → **Games** (значок игрового контроллера в левом нижнем углу)
3. Отметь галочкой **Counter-Strike: Global Offensive** или **Counter-Strike 2**
4. Перезапусти GMod

### Шаг 3 — Установи аддон

**Вариант А — полный пакет (6 ГБ, получен от автора):**
1. Распакуй архив в папку:
   ```
   GarrysMod\garrysmod\addons\cs_construct_mode\
   ```
   Должна получиться такая структура:
   ```
   addons/
   └── cs_construct_mode/
       ├── gamemodes/
       ├── lua/
       ├── models/
       ├── materials/
       ├── sound/
       └── README.md
   ```

**Вариант Б — только код (этот репозиторий):**
1. Скачай репозиторий: нажми **Code → Download ZIP**
2. Распакуй в `GarrysMod\garrysmod\addons\cs_construct_mode\`
3. Скопируй папки `models/`, `materials/models/`, `sound/` из полного пакета (у автора)

### Шаг 4 — Запуск

1. Запусти Garry's Mod
2. **Create Server** (или **Start New Game**)
3. В меню выбора геймода найди **CS Construct Mode**
4. Выбери любую карту (рекомендуется `de_dust2`, `de_inferno`, `cs_office` или другие CS карты)
5. Запусти сервер

## Управление

| Клавиша | Действие |
|---------|----------|
| **F4** | Меню закупки / выбор команды / модель / перчатки |
| **B** | Открыть меню закупки (во фриз-тайме) |
| **1–4** | Слоты оружий |
| **5** | Бомба (если выдана) |
| **Shift** | Тихий шаг (без звука шагов) |
| **Tab** | Таблица счёта |

## Команды CT-игрока
В меню закупки (F4 → вкладка **МАГАЗИН**) доступны:
- Бронежилет — 650$
- Бронежилет + шлем — 1000$
- Набор сапёра — 400$ (только CT)

## Геймплей

- **Фриз-тайм** (12 сек) → закупка, команды замёрзлые
- **Живая фаза** → стреляем, T пытаются заложить бомбу
- **Победа T:** уничтожить всех CT **или** взорвать бомбу
- **Победа CT:** уничтожить всех T **или** разминировать бомбу
- Смерть в раунде → переходишь в наблюдатель до следующего раунда
- Деньги за убийства, тратишь на закупку в следующем раунде
